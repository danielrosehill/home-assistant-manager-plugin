Review backup configuration and recent backup operations:

1. Backup Inventory
   - List all available snapshots/backups
   - Show snapshot sizes and dates
   - Identify snapshot types (full/partial)
   - Calculate total backup storage used

2. Recent Backup Activity
   - Check last successful backup date/time
   - Review recent backup operations
   - Identify any failed backups
   - Note backup duration trends

3. Backup Configuration
   - Review automated backup settings
   - Check backup frequency
   - Verify retention policy
   - Confirm backup destinations

4. Storage Analysis
   - Check backup storage space availability
   - Identify old backups for cleanup
   - Assess backup growth rate
   - Verify backup location accessibility

5. Off-Site Backup Status
   - Check if off-site backups are configured
   - Verify recent off-site backup success
   - Review cloud storage sync status
   - Confirm backup redundancy

6. Restoration Readiness
   - Verify backup integrity
   - Check if restore has been tested
   - Document restore procedure
   - Note time of last successful test restore

Provide backup health assessment and recommendations for improvement.
