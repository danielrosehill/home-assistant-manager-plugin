Check the overall health and status of the Home Assistant installation including:

1. System Information
   - Core version, Supervisor version, OS version
   - System uptime
   - Host information

2. Resource Usage
   - CPU usage
   - Memory usage
   - Disk space
   - Database size

3. Component Status
   - Core status (running/stopped)
   - Supervisor status
   - Add-on status overview
   - Integration health

4. Recent Issues
   - Check logs for recent errors or warnings
   - Identify any failed services or integrations
   - Note any performance issues

5. Connectivity
   - Network connectivity
   - Internet access
   - DNS resolution

Provide a summary assessment of system health and flag any issues requiring attention.
