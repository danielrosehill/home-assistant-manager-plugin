Review all installed add-ons and their status:

1. Add-on Inventory
   - List all installed add-ons
   - Show version information
   - Note if updates are available

2. Running Status
   - Identify which add-ons are currently running
   - Check if any have failed to start
   - Note resource usage by add-on

3. Configuration Status
   - Identify add-ons with configuration issues
   - Note any deprecated settings
   - Check for missing required configuration

4. Dependency Status
   - Check if add-on dependencies are met
   - Identify version conflicts
   - Note any compatibility issues

5. Health Assessment
   - Review add-on logs for errors
   - Check connectivity for network-dependent add-ons
   - Assess overall add-on ecosystem health

Provide recommendations for updates, fixes, or removals.
