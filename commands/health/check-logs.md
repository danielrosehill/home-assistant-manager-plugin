Analyze Home Assistant logs to identify problems:

1. Recent Errors
   - Parse home-assistant.log for recent errors
   - Identify error patterns or recurring issues
   - Note timestamps and affected components

2. Warnings
   - Review warning messages
   - Assess severity and impact
   - Recommend remediation steps

3. Component-Specific Issues
   - Integration errors
   - Add-on failures
   - Device communication problems
   - Authentication failures

4. Performance Issues
   - Slow startup warnings
   - Component load time issues
   - Database query warnings
   - Resource exhaustion messages

5. Configuration Issues
   - Deprecated features
   - Invalid configuration warnings
   - Migration notices

Provide prioritized list of issues with recommended actions.
